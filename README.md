# 老年人监护系统 - 社区端

## 1. 系统概述

本系统是一个面向社区的老年人监护平台，旨在通过智能化手段解决“空巢老人监护碎、子女异地照护难”的问题。系统将事后报警升级为主动感知与智能响应，构建集健康监测、紧急预警、服务对接于一体的智慧养老闭环。

### 核心功能
- 健康状态实时监控
- 紧急警报处理与响应
- 医生派遣工单管理
- 设备状态管理
- 巡访计划制定与执行
- AI健康风险预警
- AI辅助诊断
- 角色权限管理

## 2. 技术架构

### 前端
- HTML5 + CSS3 + JavaScript
- Bootstrap 5 (响应式设计)
- Font Awesome (图标库)
- Chart.js (数据可视化)

### 后端
- Node.js + Express
- SQLite 数据库
- Express Session (认证)
- CORS (跨域支持)

## 3. 安装部署

### 3.1 环境要求
- Node.js 14.0+ 
- npm 6.0+

### 3.2 本地部署步骤

#### 1. 安装Node.js
- 访问 https://nodejs.org 下载并安装LTS版本
- 验证安装：`node -v` 和 `npm -v`

#### 2. 获取代码
```bash
# 通过GitHub克隆
git clone https://github.com/liyunyi1226/elderly-monitoring-system.git
cd elderly-monitoring-system

# 或直接复制项目文件到本地
```

#### 3. 安装依赖
```bash
cd server
npm install
```

#### 4. 启动服务
```bash
npm start
```

#### 5. 访问系统
- 本地访问：http://localhost:3000
- 局域网访问：http://[电脑IP]:3000

### 3.3 云平台部署

#### Render部署
1. 访问 https://render.com 并登录
2. 点击 "New" → "Web Service"
3. 连接GitHub仓库
4. 配置：
   - Environment: Node
   - Root Directory: server
   - Build Command: npm install
   - Start Command: npm start
5. 点击 "Create Web Service"

#### Railway部署
1. 访问 https://railway.app 并登录
2. 点击 "New Project" → "Deploy from GitHub repo"
3. 连接GitHub仓库
4. 配置：
   - Environment: Node.js
   - Build Command: npm install
   - Start Command: npm start
   - Working Directory: server
5. 点击 "Deploy"

## 5. 使用指南

### 5.1 登录系统
1. 打开登录页面：http://localhost:3000
2. 选择角色（管理员、医生、护理员、客服）
3. 输入测试账号和密码
4. 点击 "登录系统"

### 5.2 创建派医工单
1. 登录系统后，点击 "派医工单"
2. 点击 "创建工单" 按钮
3. 选择老人、填写病情描述、选择医生
4. 点击 "创建工单" 按钮

### 5.3 处理警报
1. 点击 "警报中心"
2. 查看待处理警报
3. 点击 "处理" 按钮
4. 填写处理结果
5. 点击 "确认处理"

### 5.4 查看健康数据
1. 点击 "老年人管理"
2. 选择老人，点击 "查看详情"
3. 在 "健康数据" Tab查看详细数据

### 5.5 制定巡访计划
1. 点击 "巡访计划"
2. 点击 "创建计划"
3. 选择老人、设置巡访频率和时间
4. 点击 "保存计划"

## 7. 常见问题

### 7.1 无法登录
- 检查账号密码是否正确
- 检查服务器是否运行
- 检查网络连接

### 7.2 工单创建失败
- 检查是否选择了老人和医生
- 检查病情描述是否填写
- 查看浏览器控制台错误信息

### 7.3 设备显示离线
- 检查设备是否正常连接
- 刷新设备列表
- 检查网络连接

### 7.4 服务器启动失败
- 检查端口是否被占用
- 检查依赖是否安装成功
- 查看错误日志

### 7.5 数据库初始化失败
- 确保有写入权限
- 检查数据库文件是否被占用
- 重新启动服务
